Программа для ведения заметок 14.07.2023

Программа сформирована на языке python.

Процесс:
Пользователь используя главное меню имеет возможность полностью управлять
заметками сформированными с файле notes.csv. Файл с расширением csv может 
быть открыт через блокнот или с помощью программы Excell.
Применяемый символ ";" используется для разделения типов данных в заметке

Пользователю доступны функции просмотра, добавления, редактирования, 
удаления и поиска заметок в приложенном файле.
